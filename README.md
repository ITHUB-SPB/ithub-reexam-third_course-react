# Крестики-нолики (XO)

_(контрольная точка #2)_

## Функционал

Приложение представляет игру в крестики-нолики.

Пользователей встречает сплэш-экран с параллакс-анимацией фона. По нажатию на блок с названием появляется основной экран.

Игроки делают ходы по очереди (начинают крестики). Если один из игроков получает три-в-ряд, эти ячейки подсвечиваются зеленым, а также на зелёный меняется цвет заполнения иконки игрока (речь про Х или О над именем игрока).

При наведении на ячейку, в ней появляется размытая версия иконки текущего игрока, а при щелчке на ней иконка появляется уже без размытия. Все ходы сохраняются.

По умолчанию, игроки получают имена _Player1_ и _Player2_, но при желании могут сменить имя. Для смены имени нужно кликнуть по имени игрока - надпись изменится на поле ввода, и при появлении изменений справа появится кнопка подтверждения изменений.

По завершению игры (победой или ничьей) появляется кнопка "Начать заново", при этом имена игроков сохраняются, но порядок первого хода меняется (игравший за крестики теперь играет за нолики, и наоборот).

## Требования и критерии оценивания

### Стек

Использовать фреймворк `React 19` и сборщик `Vite 7` без дополнительных библиотек (используя лишь встроенные хуки).

### Компоненты

Определить и реализовать компоненты в соответствии с [макетом](https://www.figma.com/design/1rKJjzNZJ7BL06XaFqk8Xj/Tic-Tac-Toe--Community-?node-id=0-1&p=f&t=2bQ6SV4kGJmD35Qa-0) и следующей структурой

- экран загрузки
  - векторные иконки (анимированные)
  - основная надпись (кликабельная)
- экран игры
  - легенда (имена игроков и их иконки)
  - игровое поле
    - ячейка игрового поля
      - иконка внутри него
  - кнопка "начать снова"

Компоненты должны быть реализованы

### Хранение состояния

Состояние игры должно храниться глобально

При первом запуске оно имеет следующий вид:

```ts
{
  players: { Player1: "x", Player2: "o" },
  currentMove: null
  field: [
    [null, null, null],
    [null, null, null],
    [null, null, null]
  ],
  state: "idle"
}
```

После перехода на экран игры:

```ts
{
  players: { Player1: "x", Player2: "o" },
  currentMove: "Player1"
  field: [
    [null, null, null],
    [null, null, null],
    ["null", null, null]
  ],
  state: "progress"
}
```

После изменения имени игрока:

```ts
{
  players: { Player1: "x", Megamozg: "o" },
  currentMove: "Megamozg"
  field: [
    [null, "x", "x"],
    [null, "o", null],
    ["null", null, null]
  ],
  state: "progress"
}
```

При завершении игры:

```ts
{
  players: { Player1: "x", Megamozg: "o" },
  currentMove: "Player1"
  field: [
    ["x", "x", "x"],
    [null, "o", null],
    ["null", "o", null]
  ],
  state: "x-wins"
}
```

При перезапуске игры:

```ts
{
  players: { Player1: "o", Megamozg: "x" },
  currentMove: "Megamozg"
  field: [
    [null, null, null],
    [null, null, null],
    ["null", null, null]
  ],
  state: "progress"
}
```
